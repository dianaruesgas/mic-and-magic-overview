# Mic & Magic – Praxisprojekt (Universität Duisburg-Essen)

This repository contains a high-level description of the university project
**"Mic & Magic"**, developed as part of an interdisciplinary practical course
at the University of Duisburg-Essen.

## Project Overview
Mic & Magic is an interactive, AI-assisted system with a web-based user interface.
The project focuses on UI development, system integration, and interaction with
LLM-based backend services.

## My Responsibilities
- UI development with React, Vite, and Tailwind CSS
- State handling and notification logic
- Integration of backend interfaces
- Team collaboration using Git

## Note
The full source code is hosted in a private GitLab repository of the
University of Duisburg-Essen, a public version may be released in the future.
